Nessa aula foi abordado como se colocar simbolos e emojis nas paginas.

Para colocar os simbolos usamos codigos com o & E comercial.

Colocando os emojis.

Para colocar os emojis é necessário usar o código de cada simbolo, geramente esse código (ex do codigo: U+1F480)

Usando a EMOJIPEDIA acessaremos varios tipos.

Para adicionar os emojis usamos o & + # + x + o código do emoji e ; no final.
